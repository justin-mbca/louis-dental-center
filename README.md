# Louis Dental Center Website

Modern Next.js website for Louis Dental Center with professional digital dental laboratory copy, CAD/CAM service pages, AI dentistry concept pages, case upload UI, case tracking dashboard, and dentist portal prototype.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Main pages

- `/` — homepage
- `/services` — patient and professional services
- `/dental-lab` — digital dental lab services
- `/digital-dentistry` — CAD/CAM and digital workflow
- `/ai-dentistry` — AI-enhanced workflow concept
- `/case-upload` — frontend upload portal prototype
- `/case-tracking` — sample case tracking dashboard
- `/dentist-portal` — dentist/lab portal landing page
- `/contact` — contact form prototype
- `/faq` — frequently asked questions

## Notes

The case upload form and contact form are frontend-only prototypes. Do not use real patient information until secure upload, authentication, storage, privacy review, and backend handling are implemented.
