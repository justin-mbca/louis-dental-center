# Louis Dental Center

Modern Dentistry • Digital Dental Laboratory • CAD/CAM Solutions

## Overview

Louis Dental Center is a modern digital dentistry and dental laboratory platform focused on high-quality restorative solutions, digital workflows, and advanced CAD/CAM design services.

The platform supports dentists, dental laboratories, milling centers, and patients through digital dentistry technologies and streamlined case management workflows.

## Key Services

### Digital Dental Laboratory

* Crown & Bridge Design
* Implant Restorations
* Custom Abutments
* Full Arch Restorations
* All-on-4 / All-on-6 Workflows
* Removable Prosthetics

### Digital Dentistry

* Digital Impressions
* STL-Based Workflows
* CAD/CAM Design
* Exocad Integration
* 3Shape Integration
* Sirona Integration

### Future AI-Enhanced Features

* AI Case Intake Assistant
* Digital Case Tracking
* Dentist Portal
* AI Workflow Recommendations
* Online Case Submission
* Automated Quote Generation

## Local Development

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/louis-dental-center.git
cd louis-dental-center
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deployment

This application is optimized for deployment on Vercel.

```bash
npm run build
```
